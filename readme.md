IQ 为个体值

攻击力基础计算公式为sqrt(10)* IQ * level

防御力基础计算公式为sqrt(5)* IQ * level


NB性格提升20%特殊攻击，降低20%普通攻击

rubbish性格提升20%防御，降低20%攻击

happy性格提升10% 普通攻击 10%普通防御 降低10%特殊攻击 10%普通攻击
